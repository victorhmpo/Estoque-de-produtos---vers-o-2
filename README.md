# Estoque-de-produtos---versão-2
Versão aprimorada do programa Z34produtos_est, com novas funcionalidades, melhores práticas de programação e mais proteção ao banco de dados

Proteção contra vendas com prejuízo (custo maior ou igual ao lucro)
Tratamentos de exceção
Margem de lucro de cada produto visível na visão do estoque
Acessos e modificações mais seguros ao banco de dados
Geração automática de ID de forma segura no padrão SAP (mesmo com usos simultâneos de diferentes usuários ao sistema)
